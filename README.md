# CountingRod
## 算筹数图生成器
 - 用于生成算筹数图
 - in.txt是输入文件，最后一行必须是回车，中间间隔用一个,半角逗号
 - 生成图片在result文件夹。时间排序
 - 正数为红色，负数为黑色
 - 为了专业需要，输出图片为输入矩阵右旋90°

- in.txt文件格式：
 
```
10,583,-4745,5968
1,534,0,-12
-5885,1,854,23
37,11,-385,-548
-14,0,238,1

```
